# Food Delivery Backend (Spring Boot)
## 📌 About the Project

This project is a backend system for a Food Delivery application developed using Spring Boot. It provides RESTful APIs to manage users, restaurants, and orders.

The system allows users to register, view available restaurants, and place orders. It follows a layered architecture (Controller → Service → Repository) to ensure clean separation of concerns and maintainability.

The backend handles core functionalities like data persistence, request handling, and business logic implementation. Spring Data JPA is used for database interaction, and PostgreSQL is used as the database.

This project demonstrates real-world backend development concepts such as REST API design, CRUD operations, dependency injection, and MVC architecture.

## 🔹 Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* PostgreSQL

## 🔹 Features

* User registration & login
* Restaurant management
* Order processing
* CRUD operations

## 🔹 How to Run

1. Clone the repo
2. Configure database in application.properties
3. Run the Spring Boot application

## 🔹 Project Structure

* controller
* service
* repository
* entity
