# Food Delivery Backend (Spring Boot)

## 🔹 Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* MySQL

## 🔹 Features

* User registration & login
* Restaurant management
* Order processing
* CRUD operations

## 🔹 How to Run

1. Clone the repo
2. Configure database in application.properties
3. Run the Spring Boot application

## 🔹 Project Structure

* controller
* service
* repository
* entity
